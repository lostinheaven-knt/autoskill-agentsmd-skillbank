# Asking Better Image Questions

## Broad vs narrow questions

### Broad
Use when the user wants an overall description:
- `Describe this image in detail.`
- `What is happening in this image?`

### Narrow
Use when the user cares about a specific aspect:
- `What objects are on the table?`
- `What is the person wearing?`
- `What mood does the lighting create?`
- `Is this photo more cinematic or documentary in style?`

## Good practice

Ask only what the image can reasonably support.
Good targets include:
- visible objects
- scene layout
- actions
- clothing
- colors
- composition
- style cues

## Be careful with

- identity claims
- fine-grained text reading
- hidden/off-frame context
- guessing intent beyond visible evidence

## Practical rule

If the user’s question is vague, rewrite it into a visually answerable question before running the script.
